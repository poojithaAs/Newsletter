So I started training AI with my own scripts, my own patterns, and my company&#x27;s standards. Here&#x27;s how: Prompt 1: EBS Volume Cleanup Use this format to identify unattached EBS volumes across AWS regions. Log with timestamps. Skip deletion. Flag missing Owner/CostCenter tags.

#!/bin/bash LOG="/var/log/ebs_cleanup_$(date +%F).log" # Scan regions → Find volumes → Check tags → Log violations Prompt 2: EC2 Idle Audit "Build a script to list EC2 instances with <5% CPU usage in the last 7 days.

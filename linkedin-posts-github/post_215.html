#DAY211 Containerd CLIs: From "Why is this so hard?" to "I am the orchestrator now." When you move from Docker to containerd, no one warns you… You lose docker ps, docker run, docker logs, and even your will to live. Instead, you&#x27;re handed three mysterious tools:

• crictl

• ctr

• nerdctl And you&#x27;re left Googling like your career depends on it. Let&#x27;s decode the madness. 1. crictl: The Pod Investigator "I&#x27;m just here to check what the kubelet is doing, ma&#x27;am." What it is: A CLI tool to talk to container runtimes (via the Kubernetes CRI). Great for:

• Listing pods, containers, images on your node

• Debugging CRI-level issues (kubelet ↔ containerd)

• Running low-level commands like crictl pods, crictl inspect Not for:

• Building images, running containers like Docker crictl ps crictl inspect <container-id Tip: If your pod is failing and you can&#x27;t kubectl exec, crictl is your CSI. ctr: The Sysadmin&#x27;s Secret Weapon "You want to run containers? Sure,if you know the magic incantation." What it is: The native CLI for containerd. Extremely powerful. Zero patience.

Great for:

• Debugging containerd internals

• Low-level container lifecycle management Pain points:

• No human-friendly help messages

• Non-intuitive syntax

• No volume, build, or compose support ctr images pull https://lnkd.in/gQKYjX6R ctr run --rm -t https://lnkd.in/gQKYjX6R Pro tip: If you&#x27;re using ctr, just know: you&#x27;re in sysadmin mode now. 3. nerdctl: The Dev&#x27;s Best Friend "Hey Docker folks. I gotchu." What it is: A Docker-compatible CLI for containerd. Think: docker but for containerd.

Great for:

• Docker-style run, build, logs, exec

• compose support via nerdctl compose

• Working with K3s (which uses containerd under the hood) Why people love it:

• Familiar syntax

• Works with rootless mode

• Supports CNI plugins, BuildKit, etc. Tip: If you want Docker features without Docker running

• crictl for Kubernetes

• ctr for hardcore internals

• nerdctl for real-world dev sanity Because YAML may be our religion, but we still deserve nice things.

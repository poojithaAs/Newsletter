#WeBrokeItTogether Episode 030426: The "Fake Software Update" An employee installed a malicious update that created a backdoor.

Sara Baqla: "Why is our internal database sending traffic to an unknown IP?"

Poojitha A S: "That&#x27;s not one of ours."

Sara Baqla: "Is someone exfiltrating data?"

Poojitha A S: "Yes. And it started right after someone updated a PDF reader."

Sara Baqla: "…that wasn&#x27;t a real update, was it?"

Poojitha A S: "It was malware disguised as one."

Fix: Isolated affected machine Rotated all internal credentials Reimaged device Restricted admin install permissions

Result: Data exfiltration stopped quickly; minimal exposure.

Future Protection: Centralized patch management + remove local admin privileges.

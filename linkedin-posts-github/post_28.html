#WeBrokeItTogether RAG Series Episode 021726 : The Invisible Manual The Setup: Users are furious. They keep asking the internal bot "How do I reset my password?", and the bot keeps replying: "I cannot find any information on that." Sara is staring at the database, pointing at a document that clearly explains the process.

Sara Baqla (Dev): "I&#x27;m switching us to a bigger model. Maybe the 100 billion parameter version."

Poojitha A S (DevOps): "Budget says no. Also, why?"

Sara Baqla: "Because this model is blind! The &#x27;Password Reset&#x27; guide is right there, File #402, but the bot never sees it!"

Poojitha A S: "Sara... what is the actual title of File #402?"

Sara Baqla: "Enterprise Identity Access Management & Credential Rotation Protocols v4.2"

Poojitha A S: "Right. And the user searched for &#x27;forgot password&#x27;."

Sara Baqla: "So? It&#x27;s an AI! It should know they are the same thing!"

Poojitha A S: "Not if the vector search ranking is weak. &#x27;Credential Rotation&#x27; and &#x27;Forgot Password&#x27; are mathematically too far apart. The Retrieval step is failing before the AI even gets a chance to read the doc."

Sara Baqla: "So we rename the files?"

Poojitha A S: "No, we fix the Ranking. We need to help the search engine understand that &#x27;Identity Access&#x27; implies &#x27;Login&#x27;."

Fix: Poojitha didn&#x27;t buy a bigger model. Instead, she implemented Hybrid Search (combining Keyword search with Vector search) and added a metadata field for "tags" to the documents. They tagged the fancy file with: [&#x27;password&#x27;, &#x27;login&#x27;, &#x27;reset&#x27;, &#x27;help&#x27;].

Result: The next time a user typed "Help I can&#x27;t login," the search engine prioritized the "Credential Rotation" document because of the metadata tag, and the bot correctly answered. RAG Failure Point #2: Missing Top Ranked Documents. Vector search isn&#x27;t magic. If your corporate jargon doesn&#x27;t match user slang, the right document won&#x27;t make the cut. Optimize your indexing with metadata.
